Para ejecutar el programa de ejemplo, primero copie la dirección de la carpeta con los archivos .java y escriba en la línea de comandos
"cd" y la dirección copiada
Ejemplo: cd C:\Users\jeanb\OneDrive\Documentos\ProyectosJavaVSCode\ADA06_EstructuraDeDatos-main

Los programas de los ordenamientos y el main ya se encuentran compilados, en caso de ser necesario, para compilar el main
escriba en la línea de comandos: javac mainOrdenamiento.java 

Finalmente escriba en la línea de comandos: java mainOrdenamiento ArchivoAOrdenar.csv
Para el ordenamiento, debe seleccionar la columna del csv en el que se basara el ordenamiento y el orden, siendo 1 para orden ascendente y 2 para orden descendente

"ArchivoAOrdenar.csv" puede ser cambiado al archivo que desee ordenar
